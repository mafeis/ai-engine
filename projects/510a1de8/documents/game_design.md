## 1. 游戏概述

### 1.1 游戏类型
*   **核心类型：** 移动端策略塔防 (TD)
*   **次要类型：** 动作角色扮演 (ARPG) 元素 + 模拟经营 (轻度)

### 1.2 目标受众
*   **核心受众：** 塔防游戏爱好者，西游题材粉丝。
*   **泛受众：** 喜爱 Q 版画风、追求轻松幽默氛围的年轻玩家（“马喽”文化受众）。

### 1.3 核心玩法 (Core Loop)
玩家扮演一只“只想躺平过开心日子”的马喽（灵猴），通过在花果山及西行路上布置各种奇葩的“猴子猴孙”和“道法机关”来抵御天兵天将或各路妖魔的骚扰。
*   **局内：** 布阵 -> 激活英雄技能 -> 收集资源 -> 应对 Boss。
*   **局外：** 建设花果山 -> 升级马喽科技 -> 养成西游名将（防御塔化）。

### 1.4 独特卖点 (USP)
1.  **“马喽”文学入魂：** 游戏台词和交互充满现代打工人的自嘲与幽默感，将“西游苦差事”解构为“马喽的快乐生活”。
2.  **动态英雄塔：** 玩家控制的主角“马喽”可以在地图上自由移动，作为流动的强力防御单位，区别于传统静态塔防。
3.  **变幻莫测的“七十二变”：** 战场上可以随机掉落变身道具，让防御塔瞬间改变属性或形态，增加局内随机性（Roguelike 元素）。

## 2. 世界观背景

### 2.1 时代背景
故事发生在取经归来后的数百年。由于天界管理僵化，花果山变成了唯一的“解压胜地”。玩家作为新一代的“快乐马喽”，只想守护自己的一亩三分地，但这引起了天庭“秩序维护者”的不满。

### 2.2 主要势力
*   **马喽阵营 (玩家)：** 以主角为中心，包括各种形态的猴子（如：扔桃子的、打快板的、搬砖的）。
*   **天庭打工人 (敌对)：** 并不是邪恶的，而是为了完成“绩效”不得不来抓马喽的天兵、天将，带着浓厚的官僚主义气息。
*   **西行老友 (援军)：** 偶尔路过的唐僧（控制系）、八戒（肉盾系）、沙僧（辅助系）会作为特殊塔加入。

### 2.3 核心冲突
**“快乐的自由” vs “刻板的秩序”**。天庭认为马喽太快乐会扰乱六道轮回的严肃性，于是派兵讨伐；马喽则通过搞怪的防御方式让天兵们“破防”。

## 3. 核心系统

### 3.1 战斗系统
*   **防御塔体系：**
    *   **基础塔：** 小马喽（投石）、长矛猴（单体穿透）。
    *   **元素塔：** 炼丹炉（火）、定海神针（雷/重击）。
    *   **特殊塔：** 紧箍咒发射器（减速）、瞌睡虫（控制）。
*   **英雄系统：** 主角马喽拥有 3 个主动技能（如：猴毛分身、大闹天宫、原地躺平回血）。
*   **地形互动：** 玩家可以利用地图中的桃树、瀑布等环境元素进行防御。

### 3.2 成长系统
*   **桃园建设：** 局外经营，通过战斗获得的“仙桃”升级建筑，提升初始金币和防御塔上限。
*   **天赋树：** 分为“力量（攻击）”、“灵动（速度）”、“智慧（CD/经济）”三个分支。
*   **时装系统：** 各种搞怪的马喽皮肤（如：工装马喽、西装马喽、草裙马喽），提供微量属性加成。

### 3.3 经济系统
*   **局内货币（桃子）：** 通过击败敌人获得，用于局内建造和升级塔。
*   **局外货币（灵石）：** 核心养成货币，用于升级天赋和建筑。
*   **稀有货币（金箍棒碎片）：** 用于解锁强力英雄和稀有皮肤。

### 3.4 社交系统
*   **互助系统：** 玩家可以“借”好友的满级防御塔来应对高难度关卡。
*   **排行榜：** 基于“守护花果山天数”的异步竞技。

## 4. 技术规格

| 维度 | 规格要求 | 备注 |
| :--- | :--- | :--- |
| **目标平台** | iOS / Android / 微信小游戏 | 优先移动端适配 |
| **开发引擎** | Unity 3D (2D 渲染模式) | 保证 Q 版卡通效果的流畅度 |
| **基础分辨率** | 1920 x 1080 (16:9) | 兼容主流刘海屏，支持 UI 自适应 |
| **目标帧率** | 60 FPS | 确保技能特效触发时的丝滑感 |
| **网络要求** | 弱联网 | 核心战斗本地计算，结算及社交数据上云 |

## 5. 开发里程碑

### 阶段一：原型期 (Month 1-2)
*   完成核心战斗 Demo（1 个英雄，3 种基础塔，5 种敌人）。
*   验证“英雄自由移动”与“塔防布阵”的平衡性。

### 阶段二：内容扩张期 (Month 3-5)
*   完成前三个大章节（花果山、龙宫、天宫）的关卡设计。
*   Q 版美术风格全量产出（角色、特效、UI）。
*   接入数值成长系统和天赋体系。

### 阶段三：测试与优化期 (Month 6-7)
*   进行平衡性测试（数值调优）。
*   首轮内测，收集玩家对“马喽文化”梗的反馈。
*   性能优化，确保低端机型不掉帧。

*   多平台同步上线，开启首发活动“寻找全网最快乐马喽”。
