# 🐒 《西游世界》道具设计文档 (Item Design Document)

## 1. 道具分类 (Item Categories)

为了保证游戏的丰富度与策略性，道具分为以下五大类：

| 分类 | 描述 | 核心作用 |
| :--- | :--- | :--- |
| **神兵利器 (Weapons)** | 各种棍棒、远程投掷物或奇门兵刃。 | 提升攻击力、攻击范围与打击特效。 |
| **灵装宝衣 (Equipment)** | 头饰、披风、护甲、鞋子。 | 提升生存能力、移动速度与跳跃高度。 |
| **仙丹妙果 (Consumables)** | 各种桃子、丹药、美酒。 | 恢复生命、内力或提供短时间数值爆发。 |
| **奇珍异宝 (Key Items)** | 任务道具、通行证、特殊法宝。 | 解锁剧情、开启特定地图区域或特殊交互。 |
| **搞怪玩具 (Fun Props)** | 变身烟雾、整蛊道具。 | 纯表现力道具，增加“马喽”的快乐度。 |

## 3. 获取方式 (Acquisition Methods)

1.  **野外掉落 (Drops):**
    *   击败路边的“小妖精”可掉落普通品质的木棍、桃子。
    *   击败关卡 Boss（如黑熊精）概率掉落对应的稀有装备碎片。
2.  **蟠桃园商店 (Shop):**
    *   使用游戏货币“铜钱”购买基础消耗品。
    *   使用“仙玉”兑换限时的搞怪外观。
3.  **八卦炉制作 (Crafting):**
    *   玩家收集材料（如：玄铁、灵芝、天丝）在太上老君的八卦炉处合成高阶装备。
4.  **取经任务奖励 (Quest Rewards):**
    *   完成主线“护送唐僧”任务可获得唯一的传说级道具。

## 4. 道具经济 (Item Economy)

### 4.1 价格范围 (Currency: 铜钱)
*   **消耗品:** 50 - 200 铜钱 (轻松负担)
*   **普通装备:** 500 - 1,500 铜钱
*   **稀有装备:** 5,000 - 15,000 铜钱
*   **史诗/传说:** 无法直接购买，需通过特定活动或稀有材料兑换。

### 4.2 掉落率 (Drop Rates)
*   **普通 (白色):** 60%
*   **优秀 (绿色):** 25%
*   **稀有 (蓝色):** 10%
*   **史诗 (紫色):** 4%
*   **传说 (金色):** 1% (仅限特定 Boss 或高级宝箱)

## 5. 设计师寄语
在《西游世界》中，我们希望玩家不仅仅是为了数值而收集道具。每一个道具，无论是“歪掉的木棍”还是“闪光的金丹”，都应该在视觉上给玩家带来“Q弹、明亮、解压”的快感。让马喽的每一次挥棍，都充满快乐！
