# 《西游世界》UI 视觉设计规范文档

**版本：** 1.0
**风格：** Q版卡通 / 东方幻想 / 活泼明快

---

## 1. UI 风格定义

### 1.1 整体风格 (Overall Style)
*   **核心关键词：** 仙气十足、圆润可爱、国潮Q版、色彩明快。
*   **设计理念：** 结合中国传统艺术元素（如祥云、如意、莲花、卷轴）与现代Q版游戏的圆润感。UI 边缘应避免尖锐棱角，多采用圆角和厚实的轮廓线，营造轻松愉快的冒险氛围。
*   **质感表现：** 采用高饱和度的色块，辅以柔和的内发光和微弱的渐变，模拟“剔透玉石”或“Q弹果冻”的质感。

### 1.2 配色方案 (Color Palette)
*   **主色调（神圣金）：** `#FFD700` 用于重要边框、标题高亮。
*   **主色调（朱砂红）：** `#FF4D4D` 用于生命值、重要按钮、节日元素。
*   **辅助色（翡翠绿）：** `#2ECC71` 用于经验条、升级提示、自然元素。
*   **辅助色（琉璃蓝）：** `#3498DB` 用于法力值、水系技能、系统背景。
*   **底板色（宣纸/木纹）：** `#FDF5E6` (米白) 或 `#8B4513` (沉香色)，用于背景底纹。

### 1.3 字体规范 (Typography)
*   **标题字体：** 带有书法韵味的圆润字体（如：华康圆体、方正喵呜体），并添加金色描边或外发光。
*   **正文字体：** 简洁的圆体字（如：思源黑体 Rounded），确保在小屏幕上的可读性。

## 2. 界面列表及布局描述

### 2.1 主菜单 (Main Menu)
*   **布局：** 居中布局，背景为动态的“花果山”或“天庭”Q版全景。
*   **UI 元素：**
    *   **游戏 Logo：** 位于上方中央，字体像金灿灿的云朵，周围环绕着小小的如意金箍棒。
    *   **选项按钮：** 垂直排列在下方。按钮形状为“木质牌匾”，上面刻着“踏上旅程”、“宝库（商店）”、“修行（设置）”。
    *   **装饰物：** 两侧有漂浮的祥云，偶尔有Q版仙鹤飞过。

### 2.2 HUD (抬头显示)
*   **布局：** 环绕式分布。
*   **UI 元素：**
    *   **角色头像框（左上角）：** 一个圆形的玉石边框，内有玩家Q版头像。
    *   **血条/蓝条（紧贴头像）：**
        *   *生命值：* 一条粗大的红绸带，上面有流动的金色光效。
        *   *法力值：* 一条半透明的蓝色琉璃管，内部有气泡上升感。
    *   **技能栏（底部中央）：** 5个圆形的技能位，边框为莲花瓣形状。大招位略大，带有火焰环绕效果。
    *   **小地图（右上角）：** 边框是一个八卦阵盘，指针是一个小小的指南针。
    *   **任务追踪（左侧）：** 像一张半展开的小卷轴，边缘有轻微的磨损感。

### 2.3 背包界面 (Backpack)
*   **布局：** 左右结构。左侧为物品格，右侧为选定物品的详情预览。
*   **UI 元素：**
    *   **背景底板：** 一个深色的檀木箱子内部质感。
    *   **物品格：** 6x5 的网格，每个格子是微凹的方形玉石槽。
    *   **分类页签：** 位于背包顶部，图标分别为“丹药（葫芦）”、“装备（铠甲）”、“杂物（包裹）”。
    *   **详情框：** 背景是一张泛黄的宣纸，文字描述使用墨黑色。

### 2.4 角色信息界面 (Character Info)
*   **布局：** 居中大窗口。
*   **UI 元素：**
    *   **3D 模型展示：** 窗口中央是玩家的Q版角色，脚下踩着一朵会转动的祥云。
    *   **装备位：** 环绕在角色模型周围，图标包含：武器、头盔、衣服、鞋子、饰品。
    *   **属性面板：** 位于右侧，用小图标表示：力量（拳头）、仙法（拂尘）、防御（盾牌）。

### 2.5 设置界面 (Settings)
*   **布局：** 悬浮弹窗。
*   **UI 元素：**
    *   **滑动条：** 轨道是细长的金箍棒，滑块是一个圆滚滚的蟠桃。
    *   **开关按钮：** 开启时是亮起的“八卦灯盏”，关闭时是熄灭的灰色状态。

### 2.6 对话界面 (Dialogue)
*   **布局：** 底部横条。
*   **UI 元素：**
    *   **对话框：** 像一张完全展开的长卷轴，两端有卷轴轴柄（青铜色）。
    *   **立绘：** NPC 的Q版半身像显示在对话框左侧，会有轻微的呼吸动画。
    *   **名字标签：** 位于对话框左上方，一个带有流苏的红色小标签。

## 3. 按钮/图标设计规范

### 3.1 按钮状态 (Button States)
*   **常态 (Normal)：** 饱满的 3D 效果，带有柔和的高光。
*   **悬停/按下 (Pressed)：** 按钮整体缩小 5%，颜色变深，伴有“叮”的清脆玉石碰撞声特效。
*   **禁用 (Disabled)：** 变为石化灰，表面有裂纹效果。

### 3.2 图标设计 (Icon Specs)
*   **形状：** 所有的系统图标（设置、邮件、好友）都封装在“小灯笼”或“仙丹”形状的容器内。
*   **线条：** 统一使用 2px 的深褐色外边轮廓，增加卡通感。
