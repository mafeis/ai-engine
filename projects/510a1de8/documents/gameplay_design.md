# 《西游世界》玩法设计文档 (GDD)

## 1. 核心玩法循环 (Core Gameplay Loop)
《西游世界》采用“局内战斗+局外养成”的经典循环，强调动作打击感与策略布局的结合。

1.  **整备阶段 (Preparation):** 玩家在地图上消耗“灵气”布置各种化身塔（如：小猴精、石敢当、土地公）。
2.  **防御阶段 (Wave Defense):** 敌人（天兵天将、各路妖魔）按波次进攻。玩家操控“马喽”亲自上阵，配合塔进行连招攻击。
3.  **结算阶段 (Reward):** 完成关卡获得“仙丹”和“功德”。
4.  **成长阶段 (Progression):** 在花果山基地使用资源升级技能树、解锁新塔、强化金箍棒。

## 2. 控制方案 (Control Scheme)
为了兼顾塔防的策略性与动作的爽快感，设计如下方案：

| 输入设备 | 移动/导航 | 基础攻击/交互 | 技能释放 | 视角/塔位选择 |
| :--- | :--- | :--- | :--- | :--- |
| **键盘/鼠标** | WASD | 鼠标左键 (轻攻击) / F (交互) | Q, E, R (技能) / 1-5 (选塔) | 鼠标移动 / 滚轮缩放 |
| **触屏** | 左侧虚拟摇杆 | 右侧大按钮 (攻击) | 环绕式技能按键 | 点击空地/拖拽塔防图标 |
| **手柄** | 左摇杆 | X (轻) / Y (重) | LB, RB, RT | 右摇杆 |

## 3. 战斗系统 (Combat System)
本作区别于传统塔防，引入了深度的**动作干预机制**。

### 3.1 基础战斗
*   **攻击:** 玩家拥有轻、重攻击组合。连续点击可触发“千钧棒”连招。
*   **格挡 (Parry):** 玩家可按住格挡键。在敌人攻击瞬间格挡可触发“反弹”，将远程飞行物（如天兵的箭）反弹回去。
*   **闪避 (Dodge):** 瞬间无敌帧，成功闪避后下一次攻击附带“定身”效果。

### 3.2 连击与气势 (Combo & Qi)
*   **气势槽:** 玩家通过连击积攒气势。气势越高，场上所有“塔”的攻击频率提升（马喽的斗志感染了猴子猴孙）。
*   **连击技:** 达到一定Combo数可释放全屏清场技“大闹天宫”。

### 3.3 塔防联动
*   **共鸣:** 马喽在某个塔附近战斗时，该塔会获得增益。
*   **合击:** 某些塔（如火德星君塔）射出的火球，马喽可以用金箍棒击飞，将其转化为更强力的定向流星攻击。

## 4. 进度系统 (Progression System)

### 4.1 等级与经验
*   **修行等级 (Cultivation):** 关卡内等级，每升一级可选择一个临时天赋（Roguelike元素）。
*   **账号等级:** 永久提升马喽的基础属性。

### 4.2 解锁机制
*   **七十二变 (Tower Unlocks):** 随着剧情推进，解锁不同功能的塔。
    *   *初期:* 投石小猴（单体）、桃木桩（减速）。
    *   *中期:* 芭蕉扇（群体推退）、如意金钱豹（资源获取）。
*   **筋斗云 (Map Unlocks):** 从花果山出发，解锁天庭、龙宫、狮驼岭等主题地图。

## 5. 难度曲线 (Difficulty Curve)

*   **第一阶段 (教学期):** 单一路径，敌人为普通天兵，强调基础攻击和摆放。
*   **第二阶段 (成长期):** 引入飞行敌人（需防空塔）和重甲敌人（需马喽重击破甲）。
*   **第三阶段 (挑战期):** 多路径进攻。引入“精英怪”，会主动破坏玩家的塔，强制玩家必须去前线救援。
*   **第四阶段 (巅峰期):** Boss战。Boss拥有大范围技能，玩家需在指挥塔群的同时，通过格挡和闪避进行高强度对决。

## 6. 数值平衡 (Numerical Balance)

为了保证“快乐马喽”的体验，数值设计遵循以下原则：

| 维度 | 比例/范围定义 | 设计意图 |
| :--- | :--- | :--- |
| **伤害占比** | 玩家 (40%) : 塔 (60%) | 保证塔防是核心，但玩家操作能扭转战局。 |
| **资源获取** | 击杀奖励 + 随时间自然增长 | 防止玩家因操作失误导致经济彻底崩盘。 |
| **冷却时间 (CD)** | 基础技能 5-10s / 大招 60-90s | 维持战斗节奏，避免技能滥用。 |
| **生命值设计** | 玩家拥有自动回血机制 | 鼓励玩家去前线肉搏，减少挫败感。 |
| **塔的性价比** | 升级收益 > 铺设新塔收益 | 鼓励玩家精耕细作，优化布局。 |
